## About

Hi, I'm Mahmudi Shodmehr — the developer behind TexHub. I build practical, well-tested
tools that make other developers' work easier: framework-agnostic PHP and Laravel
packages, React and Inertia web apps, and React Native mobile apps. I care about clean
architecture, honest documentation, and shipping things that simply work. Always glad to
collaborate and build something useful together.

## Open-Source Packages

Framework-agnostic SDKs for PHP and Laravel, usable in any PHP project.

| Package | What it does |
|---------|--------------|
| [openai](https://github.com/TexhubPro/openai) | Chat, tools, vision, assistants v2, files, images, audio, embeddings |
| [telegram](https://github.com/TexhubPro/telegram) | Multi-tenant Telegram Bot API and Business — messages, media, keyboards, webhooks |
| [whatsapp-cloud-api](https://github.com/TexhubPro/whatsapp-cloud-api) | Text, media, template and interactive messages, media, webhooks |
| [instagram-graph-api](https://github.com/TexhubPro/instagram-graph-api) | OAuth and long-lived tokens, publishing, stories, comments, DMs, webhooks |
| [alif-pay](https://github.com/TexhubPro/alif-pay) | Alif Acquiring payments, tokenization, marketplace split-payments, webhooks |
| [oson-sms](https://github.com/TexhubPro/oson-sms) | OsonSMS gateway — single and bulk SMS, Bearer auth, optional signing |
| [moy-sklad](https://github.com/TexhubPro/moy-sklad) | MoySklad JSON API 1.2 — products, stock, images, orders, reports, webhooks |
| [laklak-b2b](https://github.com/TexhubPro/laklak-b2b) | LakLak B2B delivery — cities, terminals, shipments, labels, webhooks |
| [virtual-pos-andoz](https://github.com/TexhubPro/virtual-pos-andoz) | Virtual POS / fiscal cash register (OFD, Andoz) — shifts, receipts, corrections, documents, reports, counters |
| [social-auth](https://github.com/TexhubPro/social-auth) | Social OAuth login (Google and GitHub), elegant and fully tested |
| [shark-starter-kit](https://github.com/TexhubPro/shark-starter-kit) | Laravel + React (Inertia) starter with Shark UI and llms docs |
